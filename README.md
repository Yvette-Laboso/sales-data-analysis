# Sales Data Analysis Dashboard

## 📊 Project Overview
This project analyzes retail sales data to uncover insights about product performance, regional profitability, and overall business trends.

The analysis was performed using Python in Google Colab.


## 🔧 Tools Used
- Python
- pandas
- matplotlib


## 📈 Key Questions Explored
1. Which product category generated the highest sales?
2. Which region generated the highest profit?
3. How do sales vary across categories?


## 💡 Key Insights
- Technology products generated strong sales performance.
- Some regions were significantly more profitable than others.
- Data visualization helped identify business trends clearly.


## 📁 Files Included
- `sales_data_analysis.ipynb` → Main analysis notebook
- `SampleSuperstore.csv` → Dataset used for analysis


## 🚀 Skills Demonstrated
- Data cleaning
- Data analysis
- Data grouping with pandas
- Data visualization
- Business insight generation
